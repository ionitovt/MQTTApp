 MQTTApp is a Qt-\/based desktop application that illustrates the data transfer via the MQTT protocol. It allows users to publish and subscribe to topics and visualize message flow in real-\/time.\hypertarget{index_features}{}\doxysection{\texorpdfstring{Features}{Features}}\label{index_features}

\begin{DoxyItemize}
\item Real-\/time publishing and subscription to MQTT topics.
\item User-\/friendly interface for interacting with MQTT brokers.
\item Supports multiple topics and wildcard subscriptions.
\item Displays incoming messages for subscribed topics.
\item Cross-\/platform support, built using the Qt framework.
\item Doxygen support
\item Permissive license
\end{DoxyItemize}\hypertarget{index_set-up-diagram}{}\doxysection{\texorpdfstring{Set-\/up Diagram}{Set-\/up Diagram}}\label{index_set-up-diagram}

\hypertarget{index_building-the-project}{}\doxysection{\texorpdfstring{Building the project}{Building the project}}\label{index_building-the-project}
To build the project, all you need to do, {\itshape {\bfseries{after correctly \href{README.md\#Installing}{\texttt{ installing the project}}}}}, is to set up {\bfseries{Qt}}

When you start the Qt Creator and open the project you will be prompted to choose a compiler kit. Choose the kit appropriate for your system.
