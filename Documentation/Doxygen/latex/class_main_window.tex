\doxysubsection{Detailed Description}
The \doxylink{class_main_window}{Main\+Window} class extends QMain\+Window for the application\textquotesingle{}s main UI. 

This class is responsible for initializing and displaying the main window of the application. It sets up the user interface designed in the Qt Designer and manages interactions within the main window.
