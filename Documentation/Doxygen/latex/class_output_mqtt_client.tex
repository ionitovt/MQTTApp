\doxysubsection{Detailed Description}
The \doxylink{class_output_mqtt_client}{Output\+Mqtt\+Client} class extends QPlain\+Text\+Edit to provide MQTT subscription and message display capabilities. 

This class integrates MQTT subscription functionality with a QPlain\+Text\+Edit widget, allowing the application to display messages received from MQTT topics.
